### Production
- If there are some errors shown in the gui file it is due to the code editor not recognizing paths of external libraries being used , This error is resolved while compiling with the given code.
- In the root directory run the following commands in order
- javac -cp ".;mysql-connector-j-9.5.0.jar;flatlaf-3.4.1.jar;jcalendar-1.4.jar" *.java
- java -cp ".;mysql-connector-j-9.5.0.jar;flatlaf-3.4.1.jar;jcalendar-1.4.jar" Main


This project is a Student Database Management System built entirely in Java. It utilizes the following libraries and technologies:
- **Java Swing** for the graphical user interface (GUI)
- **MySQL Connector** for local database communication
- **FlatLaf** for modern UI styling

### Features
- Full CRUD (Create, Read, Update, Delete) operations
- Advanced search, modify, and filter capabilities
- Dynamic search functionality
- Export data to CSV files

This system provides an efficient and user-friendly way to manage student records.
